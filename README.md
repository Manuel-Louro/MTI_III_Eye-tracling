# Eye-tracking Data Analysis: Métodos e Técnicas de Investigação III Final Project
This project consists of a Python script developed to process, analyze, and visualize eye-tracking data collected using the **Pupil Labs Neon** glasses (Just Act Natural model). The development of this code was carried out as part of a practical assignment for the **Métodos e Técnicas de Investigação III** course at **ISPA**. The study falls within the field of **Visual Neuromarketing** and aimed to understand how basic visual characteristics, namely spatial position (left versus right) and the presence of color (salient versus neutral), influence the allocation of visual attention in decision-making and consumer contexts.

To isolate these variables under controlled conditions, the experimental task employed simple geometric stimuli (circles), varying both their color (green as the salient stimulus and gray as the neutral stimulus) and their position on the screen. The study was guided by four main hypotheses. The first (H1) proposed that the first saccade, the initial rapid eye movement, would be predominantly directed toward the left visual field when stimuli had equal salience, reflecting a spatial and cultural reading bias. The second (H2) predicted that salient-colored stimuli would attract a greater number of fixations compared to neutral stimuli. The final two hypotheses explored the interaction between these variables. H3 suggested that the combination of color and position would result in a salient-colored stimulus presented on the left being the most intensively observed of all conditions, while H4 tested whether the total fixation time would be similar between a salient stimulus presented on the right and a neutral stimulus presented on the left.

To validate these hypotheses and overcome the metric limitations of the eye-tracking equipment, it was necessary to develop a custom computational solution capable of integrating and interpreting the raw tabular data (gaze coordinates, fixations, events, and timestamps in nanoseconds) in perfect synchronization with the real-world scene video recorded by the front-facing camera of the glasses. The resulting script fulfills this objective through two main analytical approaches, replacing manual observation with rigorous data processing.

First, the script enabled the testing of the first saccade hypothesis (H1). Since the native output of the eye-tracking device only records the amplitude and velocity of these rapid eye movements, the algorithm was designed to match the start and end timestamps of the first saccade in each trial with the high-frequency gaze data file. This iterative process made it possible to recover the original pupil coordinates and mathematically calculate the angle of displacement, automatically classifying the exact direction of each participant's first visual movement.

Second, to evaluate the distribution of visual attention and the effect of color salience (H2, H3, and H4), the script functions as an automated generator of dynamic heatmaps. Through a computer vision pipeline, the algorithm overlays gaze coordinates directly onto the frames of the real-world scene video. The distinguishing feature of this approach is the use of time as a weighting factor. Rather than simply marking gaze locations in a binary manner, the system weights each coordinate according to the exact fixation duration (in milliseconds). As a result, longer fixations generate more intense heatmap regions, accurately reflecting sustained visual attention. Using a sliding temporal window, the script automatically extracts snapshots, producing a clear, objective, and chronological visual representation of attentional dynamics during the competition between the left and right visual fields and the disruptive effect of color.
